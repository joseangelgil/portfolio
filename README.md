# 🌐 Personal Portfolio

A responsive, multilingual portfolio website built to showcase my work, experience, and contact information.  
> 🖥️📱 Fully compatible with both desktop and mobile/touch devices.

---

## 🎯 Description

This portfolio was built **from scratch** as part of my personal growth as a software developer.  
It reflects my personality and skills, while offering a clean and accessible way to explore my projects.

The site includes two available languages: **English and Spanish**, easily switchable through the language selector.

---

## 🌍 Sections Included

- **Home** – A friendly welcome and brief intro
- **About Me** – Summary of my background and skills
- **Projects** – A list of featured personal projects with the technologies used in each project.
- **Contact** – links to reach out on different platforms
- **Language Selector** – Switch between **English** and **Spanish** seamlessly

---

## 💻 Technologies Used

- HTML5
- Bootstrap
- Sass (SCSS)
- JavaScript (Vanilla)

---

## ⚠️ Requirements

- Fully responsive — works perfectly on **desktop and mobile devices**
- Best experienced in **Chrome** and **Firefox**
